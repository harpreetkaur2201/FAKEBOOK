# 📘 Fakebook Project

Fakebook is a front-end social media web application built using HTML, CSS, and JavaScript.
It is designed to simulate a simplified version of a social platform where users can create posts, upload media, and view a live feed.It includes:

- A header with logo and navigation
- Post creation with text and image/video upload
- Posts feed that displays all created posts
- Modal for profile info

## 🚀Live Demo [Fakebook Web App](https://harpreetkaur2201.github.io/FAKEBOOK/)

## ✨Features

- Create posts with text and optional media
- Dynamic enabling/disabling of the Post button
- Modal for profile display
- Fully responsive layout
