# Fakebook Project

This is my **Fakebook** web application project, built with HTML, CSS, and JavaScript.  
It includes:

- A header with logo and navigation
- Post creation with text and image/video upload
- Posts feed that displays all created posts
- Modal for profile info

## Live Demo

Check out the live version of the application here:  
[Fakebook Web App](http://127.0.0.1:5500/fakebook_profile/index.html)

## Features

- Create posts with text and optional media
- Dynamic enabling/disabling of the Post button
- Modal for profile display
- Fully responsive layout
